Keyboard Shortcuts Test

Test the keyboard shortcuts for the Amazon Scraper extension.

Available Shortcuts:

  • Ctrl+Shift+S: Copy All Product Data
  • Ctrl+Shift+I: Copy Image Link
  • Ctrl+Shift+A: Copy Affiliate Link
  • Ctrl+Shift+T: Copy Product Title

Instructions:

  1. 1. Navigate to an Amazon product page
  2. 2. Use the keyboard shortcuts to test functionality
  3. 3. Check for notifications at the top of the screen
  4. 4. Verify data is copied to clipboard

Customization:

You can customize these shortcuts in the extension settings page or through Chrome's extension management page.
